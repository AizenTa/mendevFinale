Bases de la syntaxe


Variables et constantes

Les variables en JavaScript permettent de stocker et de manipuler des données. Elles sont déclarées avec les mots-clés var, let ou const.


// Déclaration de variables
let x = 5; // avec let
var y = 10; // avec var

// Déclaration de constante
const PI = 3.14159; // avec const

Les variables déclarées avec let ont une portée de bloc, tandis que celles déclarées avec var ont une portée de fonction. Les constantes déclarées avec const ne peuvent pas être réaffectées.



Types de données

JavaScript prend en charge plusieurs types de données, notamment :

  • Nombres : entiers et flottants (ex : 42, -3.14)
  • Chaînes de caractères : chaînes de texte entre guillemets simples ou doubles (ex : 'Bonjour', "Monde")
  • Booléens : valeurs logiques true ou false



let nombre = 42; let texte = 'Ceci est une chaîne'; let vrai = true; let faux = false;


Opérateurs arithmétiques et logiques

JavaScript propose une variété d'opérateurs arithmétiques et logiques, tels que :

  • Opérateurs arithmétiques : + (addition), - (soustraction), * (multiplication), / (division), % (modulo)
  • Opérateurs logiques : && (et), || (ou), ! (non)
  • Opérateurs de comparaison : == (égalité), != (inégalité), > (supérieur à), < (inférieur à), >= (supérieur ou égal à), <= (inférieur ou égal à)



let a = 5; let b = 10; let c = a + b; // 15 let d = a < b; // true let e = a > b || c > 20; // false


Structures de contrôle

Les structures de contrôle permettent de contrôler le flux d'exécution du code en fonction de certaines conditions.

  • Conditions (if, else, else if) :
let age = 18;

if (age >= 18) {
  console.log("Vous êtes majeur.");
} else {
  console.log("Vous êtes mineur.");
}


  • Boucles (for, while, do...while) :
// Boucle for
for (let i = 0; i < 5; i++) {
  console.log(i); // Affiche 0, 1, 2, 3, 4
}
// Boucle while
let j = 0;
while (j < 5) {
  console.log(j); // Affiche 0, 1, 2, 3, 4
  j++;
}


Fonctions

Les fonctions sont des blocs de code réutilisables qui peuvent accepter des paramètres et renvoyer une valeur.

// Déclaration de fonction
function additionner(a, b) {
  return a + b;
}

// Appel de fonction
let resultat = additionner(3, 5); // 8




Les fonctions peuvent également être définies comme des expressions de fonction ou des fonctions fléchées (arrow functions) :

// Expression de fonction
const multiplier = function(a, b) {
  return a * b;
};

// Fonction fléchée
const diviser = (a, b) => a / b;
