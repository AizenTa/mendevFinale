HTML Tutorial

HTML signifie "HyperText Markup Language", ce qui se traduit par "langage de balisage pour l'hypertexte". C'est un langage de balisage standard utilisé pour créer et structurer le contenu des pages web. Le HTML utilise des balises (tags) pour définir les différents éléments d'une page, tels que les titres, les paragraphes, les liens, les images, etc.

Les listes en HTML


Listes

Les listes sont un moyen pratique d'organiser et de présenter des éléments liés dans un format structuré. HTML offre deux types principaux de listes : les listes non ordonnées et les listes ordonnées. Il est également possible d'imbriquer des listes les unes dans les autres pour créer des structures hiérarchiques.


Listes non ordonnées <ul>

Les listes non ordonnées (<ul> pour "unordered list") sont utilisées pour représenter un groupe d'éléments sans ordre particulier. Chaque élément est précédé d'une puce par défaut.




HTML :

<ul> <li>Pomme</li> <li>Banane</li> <li>Orange</li> </ul>


Listes ordonnées <ol>

Les listes ordonnées (<ol> pour "ordered list") sont utilisées pour représenter un groupe d'éléments dans un ordre spécifique, généralement numéroté.

HTML :

<ol> <li>Étape 1</li> <li>Étape 2</li> <li>Étape 3</li> </ol>


Listes imbriquées

Il est possible d'imbriquer des listes les unes dans les autres pour créer des structures hiérarchiques. Une liste imbriquée est une liste placée à l'intérieur d'un élément li d'une autre liste.


HTML :
<ul> <li>Fruits <ul> <li>Pomme</li> <li>Banane</li> </ul> </li> <li>Légumes <ol> <li>Carotte</li> <li>Brocoli</li> </ol> </li> </ul>



Description

Dans cet exemple, nous avons une liste non ordonnée principale avec deux éléments. Le premier élément contient une liste non ordonnée imbriquée de fruits, et le deuxième élément contient une liste ordonnée imbriquée de légumes.


Vous pouvez contrôler le type de puce ou de numérotation utilisé pour chaque liste en utilisant les attributs type ou start. Par exemple, ol type="A" créera une liste ordonnée avec des lettres majuscules.


Les listes sont très utiles pour présenter des informations de manière structurée et hiérarchique, que ce soit pour des instructions, des menus, des descriptions ou tout autre contenu organisé.
