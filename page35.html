Modules et ES6+


Modules en JavaScript

Les modules sont un moyen d'organiser et de partager le code en JavaScript. Avant l'introduction des modules natifs avec ES6 (ECMAScript 2015), les développeurs utilisaient des solutions tierces comme CommonJS (Node.js) ou AMD (RequireJS).
Avec ES6, les modules sont désormais pris en charge nativement dans les navigateurs modernes et Node.js. Voici comment utiliser les modules :


// math.js
export const PI = 3.14159;

export function calculateCircumference(radius) {
  return 2 * PI * radius;
}

// app.js
import { PI, calculateCircumference } from './math.js';

console.log(PI); // 3.14159
console.log(calculateCircumference(5)); // 31.4159

Les modules permettent de mieux organiser le code, d'éviter les conflits de noms et de charger uniquement les fonctionnalités dont on a besoin.


Nouvelles fonctionnalités ES6+

ES6 (ECMAScript 2015) et les versions ultérieures ont introduit de nombreuses nouvelles fonctionnalités pour rendre le développement en JavaScript plus moderne et plus efficace.


Let et const

let et const sont de nouveaux mots-clés pour déclarer des variables. const est utilisé pour déclarer des constantes (valeurs immuables), tandis que let est utilisé pour déclarer des variables dont la valeur peut changer.


Fonctions fléchées (arrow functions)


Les fonctions fléchées offrent une syntaxe plus concise pour définir des fonctions et lient automatiquement la valeur de this au contexte englobant.


const double = x => x * 2; // Fonction fléchée sans accolades const greet = (name) => { console.log(`Bonjour ${name} !`); }



Classes


ES6 introduit une syntaxe de classes pour le JavaScript, facilitant la programmation orientée objet.


class Person { constructor(name) { this.name = name; } greet() { console.log(`Bonjour, je suis ${this.name}`); } } const john = new Person('John'); john.greet(); // Bonjour, je suis John


Déstructuration

La déstructuration permet d'extraire facilement des valeurs d'objets ou de tableaux.


const person = { name: 'John', age: 30 }; const { name, age } = person; console.log(name); // John console.log(age); // 30 const [first, second] = ['apple', 'banana']; console.log(first); // apple console.log(second); // banana


Opérateur de propagation (...)

L'opérateur de propagation permet de copier les éléments d'un tableau ou les propriétés d'un objet dans un nouveau tableau ou objet.


const numbers = [1, 2, 3];
const newNumbers = [...numbers, 4, 5];
console.log(newNumbers); // [1, 2, 3, 4, 5]

const person = { name: 'John', age: 30 };
const newPerson = { ...person, city: 'Paris' };
console.log(newPerson); //

ES6+ a apporté de nombreuses nouvelles fonctionnalités pour rendre le JavaScript plus moderne, plus lisible et plus expressif. Les modules, les fonctions fléchées, les classes, la déstructuration et l'opérateur de propagation ne sont que quelques exemples parmi les nouveautés introduites.
