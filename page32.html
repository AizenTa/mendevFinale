Manipulation du DOM

Le DOM (Document Object Model) est une interface de programmation qui représente la structure d'une page web sous forme d'un arbre d'objets. JavaScript permet de manipuler le DOM, ce qui permet de créer des pages web dynamiques et interactives.


Sélection d'éléments

Avant de pouvoir manipuler les éléments du DOM, il faut d'abord les sélectionner. Voici quelques méthodes couramment utilisées :

// Sélectionner un élément par son identifiant const monElement = document.getElementById('monId'); // Sélectionner des éléments par leur nom de balise const paragraphes = document.getElementsByTagName('p'); // Sélectionner des éléments par leur classe const elements = document.getElementsByClassName('maClasse'); // Sélectionner le premier élément correspondant à un sélecteur CSS const premierElement = document.querySelector('.maClasse'); // Sélectionner tous les éléments correspondant à un sélecteur CSS const tousElements = document.querySelectorAll('div p');


Accès et modification des propriétés des éléments

Une fois qu'un élément est sélectionné, on peut accéder et modifier ses propriétés, comme son contenu textuel, ses attributs ou son style CSS.


// Accéder au contenu textuel d'un élément const paragraphe = document.querySelector('p'); console.log(paragraphe.textContent); // Affiche le texte du paragraphe // Modifier le contenu textuel d'un élément paragraphe.textContent = 'Nouveau texte'; // Accéder à un attribut d'un élément const lien = document.querySelector('a'); console.log(lien.href); // Affiche l'URL du lien // Modifier un attribut d'un élément lien.href = 'https://nouvelleurl.com'; // Modifier le style CSS d'un élément const div = document.querySelector('div'); div.style.backgroundColor = 'red'; div.style.padding = '10px';


Création et suppression d'éléments

JavaScript permet également de créer de nouveaux éléments HTML et de les ajouter ou supprimer du DOM.


// Créer un nouvel élément const nouveauParagraphe = document.createElement('p'); nouveauParagraphe.textContent = 'Nouveau paragraphe'; // Ajouter un élément au DOM const container = document.querySelector('#container'); container.appendChild(nouveauParagraphe); // Supprimer un élément du DOM const elementASupprimer = document.getElementById('elementASupprimer'); elementASupprimer.parentNode.removeChild(elementASupprimer);


Gestion des événements

JavaScript permet de réagir à différents événements tels que les clics de souris, les frappes au clavier ou le défilement de la page. Cela permet de créer des interactions riches avec l'utilisateur.

Voici quelques événements courants :

  • click : clic de souris
  • mouseover : survol avec la souris
  • keydown : touche du clavier enfoncée
  • scroll : défilement de la page
  • submit : envoi d'un formulaire




// Ajouter un gestionnaire d'événement
const bouton = document.querySelector('button');
bouton.addEventListener('click', function() {
  console.log('Le bouton a été cliqué !');
});

// Supprimer un gestionnaire d'événement
bouton.removeEventListener('click', fonctionGestionnaire);

La manipulation du DOM avec JavaScript ouvre de nombreuses possibilités pour créer des interfaces web riches et interactives. En sélectionnant, modifiant et créant des éléments, ainsi qu'en gérant les événements, vous pouvez donner vie à votre site web.
