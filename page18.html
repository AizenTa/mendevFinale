CSS Tutorial

 CSS signifie "Cascading Style Sheets" (feuilles de style en cascade).
C'est un langage de style utilisé pour décrire l'apparence et la présentation des documents HTML.
Il permet de contrôler des aspects tels que la mise en forme du texte, la disposition des éléments, les couleurs, les images d'arrière-plan, les animations, etc.
Le CSS est un standard web maintenu par le World Wide Web Consortium (W3C).

Styles de texte

Polices

  • font-family : Définit la famille de polices à utiliser (serif, sans-serif, monospace, etc.). Vous pouvez spécifier plusieurs polices en les séparant par des virgules, ce qui permet au navigateur d'utiliser la première police disponible. Exemple : font-family: Arial, sans-serif;


  • font-style : Définit le style de police (normale, italique, oblique). Exemple : font-style: italic;


  • font-weight : Définit la graisse de la police (normal, bold, lighter, bolder, ou valeurs numériques). Exemple : font-weight: bold;


  • font-size : Définit la taille de la police (en pixels, em, rem, pourcentage, etc.). Exemple : font-size: 16px;

    • Couleurs

  • color : Définit la couleur du texte.


  • Couleurs nommées : red, blue, green, etc.


  • Codes hexadécimaux : #FF0000 (rouge), #00FF00 (vert), #0000FF (bleu).


  • Valeurs RGB : rgb(255, 0, 0) (rouge), rgb(0, 255, 0) (vert), rgb(0, 0, 255) (bleu).


  • Valeurs RGBa : Similaire à RGB, mais avec un quatrième paramètre pour l'opacité (rgba(0, 0, 0, 0.5)).

    • Alignement et espacement

  • text-align : Aligne le texte horizontalement (gauche, droite, centre, justifié).

  • text-indent : Indente le premier ligne d'un paragraphe.


  • letter-spacing : Définit l'espacement entre les caractères.


  • word-spacing : Définit l'espacement entre les mots.


  • line-height : Définit la hauteur de ligne du texte.

    • Décoration

  • text-decoration : Ajoute une décoration au texte (souligné, barré, soulignement en pointillés, etc.).


  • text-transform : Transforme la casse du texte (uppercase, lowercase, capitalize).


  • text-shadow : Ajoute une ombre au texte.
