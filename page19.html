CSS Tutorial

 CSS signifie "Cascading Style Sheets" (feuilles de style en cascade).
C'est un langage de style utilisé pour décrire l'apparence et la présentation des documents HTML.
Il permet de contrôler des aspects tels que la mise en forme du texte, la disposition des éléments, les couleurs, les images d'arrière-plan, les animations, etc.
Le CSS est un standard web maintenu par le World Wide Web Consortium (W3C).

Styles de boîtes

Modèle de boîte

  • Chaque élément HTML est considéré comme une boîte rectangulaire.


  • Le modèle de boîte CSS décrit la disposition des différentes parties de cette boîte : le contenu (content), le padding (espace interne), la bordure (border) et la marge (margin).


  • Le calcul des dimensions d'une boîte dépend de la propriété box-sizing (content-box ou border-box).

    • Marges et padding

  • margin : Définit la marge extérieure de la boîte, c'est-à-dire l'espace entre la bordure et les éléments environnants.


  • padding : Définit la marge intérieure de la boîte, c'est-à-dire l'espace entre le contenu et la bordure.


  • Ces propriétés peuvent être définies individuellement (margin-top, padding-left, etc.) ou regroupées (margin: 10px 20px;).

    • Bordures

  • border-style : Définit le style de la bordure (solid, dotted, dashed, etc.).


  • border-width : Définit l'épaisseur de la bordure.


  • border-color : Définit la couleur de la bordure.


  • Vous pouvez définir les styles de bordure individuellement (border-top-style, border-right-width, etc.) ou les regrouper (border: 1px solid black;).

    • Dimensions

  • width : Définit la largeur de la boîte.

  • height : Définit la hauteur de la boîte.


  • max-width et max-height : Définissent les dimensions maximales de la boîte.


  • min-width et min-height : Définissent les dimensions minimales de la boîte.
