Commencez

Qu'est-ce que JavaScript ?

JavaScript est un langage de programmation de scripts côté client utilisé principalement pour créer des pages web interactives et dynamiques. Il a été créé en 1995 par Brendan Eich chez Netscape Communications Corporation. JavaScript n'a aucun lien direct avec le langage de programmation Java, malgré la similarité des noms.

Rôle de JavaScript dans le développement web


JavaScript permet d'ajouter des fonctionnalités interactives et dynamiques à une page web. Voici quelques exemples de son utilisation :


  • Manipulation du contenu HTML et du style CSS d'une page
  • Validation de formulaires côté client
  • Créer des animations, des effets de mouvement et des menus interactifs
  • Développer des applications web monopage (Single Page Applications)
  • Interagir avec les API du navigateur (géolocalisation, caméra, microphone, etc.)


Fonctionnement de JavaScript dans le navigateur

Lorsqu'un utilisateur accède à une page web contenant du code JavaScript, le navigateur web charge d'abord le contenu HTML et CSS. Ensuite, le moteur JavaScript du navigateur (comme V8 dans Google Chrome) interprète et exécute le code JavaScript.


Le code JavaScript peut être inclus directement dans le fichier HTML à l'aide de la balise <script>, ou il peut être chargé à partir d'un fichier externe avec l'extension .js.



Exemple d'inclusion de JavaScript dans un fichier HTML :

<!DOCTYPE html>
<html>
  <head>
    <title>Mon site web</title>
  </head>
  <body>
    <h1>Bienvenue sur mon site !</h1>
    <p id="message">Ce message sera modifié par JavaScript.</p>

    <script>
      // Code JavaScript
      const messageElement = document.getElementById('message');
      messageElement.textContent = 'Le message a été modifié !';
    </script>
  </body>
</html>

Dans cet exemple, le code JavaScript sélectionne un élément HTML avec l'ID message et modifie son contenu textuel.
