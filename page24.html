CSS Tutorial

 CSS signifie "Cascading Style Sheets" (feuilles de style en cascade).
C'est un langage de style utilisé pour décrire l'apparence et la présentation des documents HTML.
Il permet de contrôler des aspects tels que la mise en forme du texte, la disposition des éléments, les couleurs, les images d'arrière-plan, les animations, etc.
Le CSS est un standard web maintenu par le World Wide Web Consortium (W3C).

Les sélecteurs avancés


Voici un exemple général :


    HTML :

<form> <label for="name">Nom :</label> <input type="text" id="name" name="name" required> <label for="email">Email :</label> <input type="email" id="email" name="email"> <button type="submit">Envoyer</button> </form> <nav> <ul> <li><a href="#">Accueil</a></li> <li><a href="#">Produits</a></li> <li><a href="#">Contact</a></li> </ul> </nav> <article> <h2>Titre de l'article</h2> <p>Ceci est le premier paragraphe de l'article. <span>Sélectionnez ce texte</span> pour voir l'effet de la pseudo-classe ::selection.</p> <p>Ceci est le deuxième paragraphe de l'article.</p> </article>


CSS :

/* Sélecteurs d'attributs */ input[required] { border: 2px solid #ff6347; } input[type="email"] { width: 200px; } /* Sélecteurs de pseudo-classes */ a:hover { color: #ff6347; } button:active { background-color: #f0f0f0; } li:nth-child(2) { font-weight: bold; } /* Sélecteurs de pseudo-éléments */ h2::before { content: "// "; color: #777; } p::first-letter { font-size: 1.5em; font-weight: bold; } ::selection { background-color: #ff6347; color: #fff; }


Sélecteurs d'attributs


Les sélecteurs d'attributs permettent de cibler des éléments HTML en fonction de la présence, de la valeur ou de la partie d'une valeur d'un attribut spécifique. Quelques exemples:

  • [attribut]: Cible les éléments qui possèdent l'attribut spécifié, quelle que soit sa valeur.
  • [attribut="valeur"]: Cible les éléments dont l'attribut a la valeur exacte spécifiée.
  • [attribut^="valeur"]: Cible les éléments dont l'attribut commence par la valeur spécifiée.
  • [attribut$="valeur"]: Cible les éléments dont l'attribut se termine par la valeur spécifiée.
  • [attribut*="valeur"]: Cible les éléments dont l'attribut contient la valeur spécifiée.

Ces sélecteurs sont utiles pour styliser des éléments en fonction de leur type (input, button, etc.), de leur état (checked, disabled, etc.) ou de tout autre attribut personnalisé.


Sélecteurs de pseudo-classe

Les pseudo-classes permettent de cibler des états spécifiques d'un élément. Certaines pseudo-classes courantes sont:

  • :hover: Cible l'élément lorsque le curseur de la souris passe au-dessus.
  • :active: Cible l'élément lorsqu'il est activé (par exemple, un bouton enfoncé).
  • :focus: Cible l'élément lorsqu'il reçoit le focus (par exemple, un champ de formulaire).
  • :nth-child(n): Cible les éléments enfants spécifiques en fonction de leur position.
  • :nth-of-type(n): Cible les éléments spécifiques en fonction de leur type et de leur position parmi les éléments du même type.

Les pseudo-classes permettent de créer des styles dynamiques qui changent en fonction de l'état ou de l'interaction de l'utilisateur avec l'élément.


Sélecteurs de pseudo-élément

Les pseudo-éléments permettent de cibler et de styler des parties spécifiques d'un élément. Voici quelques exemples courants:

  • ::before: Ajoute un contenu avant l'élément cible.
  • ::after: Ajoute un contenu après l'élément cible.
  • ::first-line: Cible la première ligne de texte d'un élément.
  • ::first-letter: Cible la première lettre de texte d'un élément.
  • ::selection: Cible la partie de texte sélectionnée par l'utilisateur.

Les pseudo-éléments sont utiles pour ajouter des ornements, des icônes ou des effets décoratifs avant ou après un élément, ou pour styliser spécifiquement les premières lignes ou lettres d'un texte.
