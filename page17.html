CSS Tutorial

 CSS signifie "Cascading Style Sheets" (feuilles de style en cascade).
C'est un langage de style utilisé pour décrire l'apparence et la présentation des documents HTML.
Il permet de contrôler des aspects tels que la mise en forme du texte, la disposition des éléments, les couleurs, les images d'arrière-plan, les animations, etc.
Le CSS est un standard web maintenu par le World Wide Web Consortium (W3C).

Syntaxe de base du CSS


Règles CSS

  • Une règle CSS est composée d'un sélecteur suivi d'un bloc de déclarations entre accolades 0.
  • Exemple : h1
  • Les déclarations à l'intérieur des accolades définissent les styles à appliquer aux éléments ciblés par le sélecteur.

    • Sélecteurs

  • Les sélecteurs permettent de cibler les éléments HTML auxquels les styles CSS doivent être appliqués.
  • Types d'éléments : div, h1, p, etc. (sélectionnent tous les éléments de ce type).
  • Classes : .nomClasse (sélectionne tous les éléments ayant cette classe).
  • ID : #idElement (sélectionne l'élément unique avec cet identifiant).
  • Sélecteurs universels : * (sélectionne tous les éléments).
  • Sélecteurs descendants : div p (sélectionne tous les éléments <p> à l'intérieur de <div>).
  • Sélecteurs enfants : ul > li (sélectionne les éléments <li> directement enfants de <ul>).
  • Vous pouvez combiner les sélecteurs pour cibler des éléments spécifiques.

    • Propriétés et

    valeurs

  • Une déclaration CSS est composée d'une propriété et d'une valeur, séparées par deux-points :.
  • Exemples de propriétés : color, font-size, margin, background-color, etc.
  • Chaque propriété accepte des valeurs spécifiques (couleurs, tailles, unités, mots-clés, etc.).
  • Plusieurs déclarations sont séparées par un point-virgule ;.

    • Commentaires

  • Les commentaires en CSS permettent d'ajouter des annotations et des explications dans le code.
  • Un commentaire en CSS commence par /* et se termine par */.
  • Exemple : /* Ce commentaire explique la règle CSS suivante */
  • Les commentaires sont ignorés par les navigateurs et n'affectent pas le rendu des styles.

    Ce sont les bases de la syntaxe CSS que vous devez maîtriser avant d'explorer des concepts plus avancés.


    <!-- HTML --> <!DOCTYPE html>


    <html>
               <head>
                <title>Exemple CSS</title>
                    <style> 

                  * Règle CSS ciblant tous les éléments de titre h1 */
                       h1 { color: navy; font-family: Arial, sans-serif;
                     }
            /* Règle CSS ciblant les éléments de paragraphe avec la classe "important" */
                         p.important { font-weight: bold; font-size: 16px;
                            }
                    </style>
              </head>
                  <body>
              <h1>Bienvenue sur mon site</h1>
              <p>Ceci est un paragraphe normal.</p>
              <p class="important">Ceci est un paragraphe important.</p>
        </body>
    </html>
