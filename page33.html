Objets et tableaux

Création d'objets

Les objets en JavaScript permettent de stocker des collections de données et de les structurer. Ils sont composés de paires clé-valeur appelées propriétés.

// Objet littéral const personne = { nom: 'Dupont', age: 30, adresse: { rue: '10 rue de la Paix', ville: 'Paris' } }; // Objet avec constructeur function Voiture(marque, modele, annee) { this.marque = marque; this.modele = modele; this.annee = annee; } const maCar = new Voiture('Toyota', 'Corolla', 2018);


Accès et modification des propriétés d'un objet


Vous pouvez accéder et modifier les propriétés d'un objet en utilisant la notation par points ou la notation par crochets.


// Accès aux propriétés console.log(personne.nom); // 'Dupont' console.log(personne['adresse']['ville']); // 'Paris' // Modification des propriétés personne.age = 31; personne['adresse']['rue'] = '20 rue de la Paix';


Tableaux (création, accès, modification)


Les tableaux en JavaScript sont utilisés pour stocker et manipuler des collections ordonnées de valeurs.


// Création d'un tableau const fruits = ['pomme', 'banane', 'orange']; const nombres = [1, 2, 3, 4, 5]; // Accès aux éléments d'un tableau console.log(fruits[0]); // 'pomme' console.log(nombres[3]); // 4 // Modification d'un élément du tableau fruits[1] = 'fraise';


Boucles et itération sur les tableaux

Vous pouvez itérer sur les éléments d'un tableau en utilisant une boucle for, une boucle for...of ou une méthode comme forEach().


// Boucle for classique for (let i = 0; i < fruits.length; i++) { console.log(fruits[i]); } // Boucle for...of for (const fruit of fruits) { console.log(fruit); } // Méthode forEach() nombres.forEach(function(nombre) { console.log(nombre); });


Vous pouvez également utiliser d'autres méthodes courantes pour manipuler les tableaux, comme push(), pop(), slice(), map(), filter() et reduce().


// Ajouter un élément à la fin du tableau fruits.push('poire'); // Supprimer le dernier élément du tableau const dernierFruit = fruits.pop(); // Créer un nouveau tableau avec une partie des éléments const quelquesFruits = fruits.slice(1, 3); // Créer un nouveau tableau avec des valeurs transformées const doublesNombres = nombres.map(function(nombre) { return nombre * 2; }); // Créer un nouveau tableau avec des éléments filtrés const nombresImpairs = nombres.filter(function(nombre) { return nombre % 2 !== 0; });
